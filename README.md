# 风语 · Winsper

2019 夏季小学期项目管理及软件开发课程设计项目

## 项目介绍

### 定位

一款专注于记录心情、分享生活的校园社交APP

### 架构

+ React Native 前端
+ Spring Cloud 后端
+ Drone + Docker 运维

## 目录结构

+ `Winsper-*` 微服务子模块
+ `API-Spec` 前后端接口规约
+ `doc` 项目文档
